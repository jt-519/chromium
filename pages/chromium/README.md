## 准出标准速查

### P1 — 性能基准

| 指标 | 通过条件 | 阻断条件 | 级别 |
|------|---------|---------|------|
| Speedometer 3.0 | 新核 >= 旧核 95% | R < 0.90 | P0 |
| MotionMark | 新核 >= 旧核 90% | R < 0.80 | P0 |
| WebGL 1000鱼 | FPS >= 旧核 90% | R < 0.80 | P1 |
| WebGL 衰减率 | <= 旧核 + 10pp | 差值 > 20pp | P1 |
| JetStream 2 | 新核 >= 旧核 90% | R < 0.85 | P1 |

### P2 — 兼容性

| 指标 | 通过条件 | 阻断条件 | 级别 |
|------|---------|---------|------|
| Kernel Test API 通过率 | >= 99.5% 且新增 Fail = 0 | < 98% 或无豁免新增 Fail | P0 |
| Kernel Test 性能回归 | 劣化 <= 5% | 劣化 > 15% | P0 |
| Browser Score | >= 旧核分数 | 非豁免退化项 > 0 | P0 |
| Breaking Changes | 全部已评估+验证 | 影响业务且无预案 | P0 |

### P3 — 业务页面

| 指标 | 通过条件 | 级别 |
|------|---------|------|
| FCP / LCP | <= 旧核 110% | P0 |
| CLS | < 0.1 | P1 |
| JS Bridge | 全部正常 | P0 |
| 内存峰值 | <= 旧核 120% | P1 |

### P4 — 稳定性

| 指标 | 通过条件 | 级别 |
|------|---------|------|
| Crash 率 | <= 对照 | P0 |
| ANR 率 | <= 对照 | P0 |
| Monkey 50K | 0 Crash | P0 |

## 测试环境规范

- 关闭所有扩展/后台应用；飞行模式 + 屏幕常亮 + 亮度 50%
- 设备预热 > 5 分钟，浏览器冷启动后空闲 30 秒再开测
- 每项基准跑 **3 次**，取中位数
- 同设备跨版本对比时，同一时段交替测试
- 波动容差：同环境 ±3%；跨日期 ±5%

## 后续升核 (M141+)

本流程适用于任意跨版本升核。下次升级时：

1. 更新 `kernel_upgrade_diff.html` 数据（新增差异条目）
2. 更新 `kernel_changelog.html` 数据（补充新版本变更）
3. 使用本 README 模板执行完整 P1-P4 流程
4. 填写测试报告提交评审
