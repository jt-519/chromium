如何按内核版本下载编译产物
  1. 打开 Chromium Dash → Releases (Android)，找到目标版本号（如 141.0.7359.0）
  2. 点击版本号进入详情页，记录 Chromium Commit Position（如 1456789）
  3. 或直接用 API 查询：https://chromiumdash.appspot.com/fetch_releases?channel=Stable&platform=Android&num=10，响应中 chromium_main_branch_position 即为 commit position
  4. 打开 编译产物快照，选择平台目录（Android → Android/）
  5. 在搜索框输入 commit position 数字，进入对应目录下载 chrome-android.zip 或 system_webview_google-x86.apk
  6. 若精确 position 不存在，取最近的可用 position（通常 ±1~3 范围内有快照）
